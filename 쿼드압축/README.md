## 쿼드 압축
> 프로그래머스 레벨2

### 풀이
※ 백준 쿼드트리 문제와 매우 매우 흡사한 문제였다.
1. 분할 정복으로 푼다.
   1. 4면으로 나눈다.
      - 행, 열이  
       - 행: 처음~중간, 열: 처음~중간 
       - 행: 처음~중간, 열: 중간~끝
       - 행: 중간~끝, 열: 처음~중간
       - 행: 중간~끝, 열: 중간~끝
    2. 각각의 면을 탐색한다. 
       - 만약 현재 탐색하는 면에 0, 1이 혼재되어있다면? false
       - 모두 0 혹은 모두 1이라면? true 
         - 이때 0이라면 0의 개수++
         - 이때 1이라면 1의 개수++ 
         - 해당 면적을 모두 방문했음 표시해준다.
2. 1번에서 세어준 0과 1의 개수를 배열화하여 리턴한다.
